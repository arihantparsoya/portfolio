Telecom ParisTech

I explored user scenarios where transclusions with histories can be useful and created tool which can be used to explore histories of transclusion. I used Webstrates technology to implement tools for exploring transclusion histories. [Report]

Google Summer of Code 2017
Python Software Foundation, May - Aug '17

I worked for SymPy organization during my Google Summer of Code internship. My work focused on implementing Rubi (Rule Based Integration) algorithms in SymPy which are more faster and accurate then generic symbolic integration algorithms. I used MatchPy library to enable advanced pattern matching capabilities in SymPy and created parsers which was capable of converting Mathematica code to Python. [Link]

Transitions in Interaction Design
2018

A research project done under Prof. Venkatesh Rajamanickam. The project focused on understanding transitions and how they can be used as cognitive tool in interface design. I looked at research done in animation and Psychology of Motion Perception to understand how transitions can be used effectively and proposed a new set of principles which can be used by designers to effectively use transitions while designing interface.
